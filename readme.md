# Projeto Time de Futebol
# JS com BS e Module
Montando front-end com arquivo de dados no formato de
array de objtos

# Campos do Objeto

| Campo            | Tipo     | Descrição |
|------------------|----------|------------|
| `agilidade`      | `number` | Nível de agilidade do jogador (50 a 100). |
| `altura`         | `number` | Altura do jogador em centímetros. |
| `clube_atual`    | `string` | Nome do clube atual do jogador. |
| `codigo_jogador` | `number` | Identificador único do jogador. |
| `drible`         | `number` | Habilidade de drible do jogador (50 a 100). |
| `forca`          | `number` | Nível de força física do jogador (50 a 100). |
| `foto`           | `string` | Caminho/nome do arquivo da imagem do jogador. |
| `idade`          | `number` | Idade atual do jogador. |
| `nacionalidade`  | `string` | País de origem do jogador. |
| `nome`           | `string` | Nome completo do jogador. |
| `peso`           | `number` | Peso do jogador em quilogramas. |
| `posicao`        | `string` | Posição principal do jogador em campo. |
| `velocidade`     | `number` | Nível de velocidade do jogador (50 a 100). |
